Лични проекти

  • Code Hub

    Code Hub е колаборативен редактор на код, създаден с цел да улесни обучението по програмиране. Той позволява на учениците в реално време да виждат какво прави техният учител, както и обратното. Така те могат да учат без да се налага да си извиват вратовете по всевъзможни начини само за да видят дъската, а после като дойде време за упражненията, учителят може да наблюдава кой пише, кой не, кой има затруднения със някоя задача и може да му/ѝ помогне без да става от компютъра.
    Github: Тук

  • World Cup Prediction
